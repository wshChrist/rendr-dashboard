- `ea/RendrAccountMonitor.mq4` - EA de référence (fonctionne)

## Prochaines étapes

Créer une nouvelle EA propre pour RendR qui :
1. S'enregistre auprès de l'API lors de l'initialisation
2. Envoie les trades en temps réel
3. Gère les erreurs de connexion

## Configuration requise

- MetaTrader 4 ou 5
- URL de l'API à configurer dans l'EA
- Liste blanche MetaTrader configurée
